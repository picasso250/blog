PHP 文档读后感
=================

文档犹如字典，闲暇时看的人不多。

PHP的文档很有用，自不必说，底下的评论里是很有些干货的。

基本语法
----------

PHP 标签闭合处相当于自动添加引号。所以下面两个句子完全等同。

```php
<?php echo 42; ?>
<?php echo 42 ?>
```

如果一个文件全部是由PHP构成的，比如类的定义啥的，那么就一定不要添加闭合标签。

正确：

```php
<?php
class Foo
{
}
```

错误：
```php
<?php
class Foo
{
}
?>
```

省略闭合标签的是为了避免意外的输出。
